
# RENTIFY

Tech Stack

     Front end: Angular,
                Angular Material.

     Back end: NodeJS,
               ExpressJs.

     Database : MangoDB.


RUNNING THE APPLICATION:

     Front end command: 
        npm install
        ng serve
        
     Back end:
        MangoDB must be installed
        create a Database named 'rentify'
        run npm install
        run npm start
